# profile-card-project
Este é um projeto de Cartão de Links desenvolvido para praticar fundamentos de Front-End. O layout foi inspirado em um design moderno.

🚀 Tecnologias Utilizadas
- HTML5: Estrutura semântica (uso de tags `<a>` para links).
- CSS3: Layout com Flexbox, estilização de botões pílula e ícones circulares.
- FontAwesome: Ícones das redes sociais.
- Google Fonts: Fontes Great Vibes e Montserrat.

💡 Aprendizados
Durante o desenvolvimento, foquei em:
- Alinhamento perfeito de ícones usando Flexbox.
- Diferença entre elementos de bloco e inline.
- Reset de estilos padrão do navegador (como a cor dos links).
